# `IO`类型
 - `Item`
 - `Fluid`
 - `Energy`

---

# 定义 `IO` 模式
 - 通过方法重写的方式来选择每种类型的 `IO`
 - 通过返回 `IOMode` 来定义该类型的输入输出模式

---

# `IOMode` 说明
 - `INPUT`：仅输入
 - `OUTPUT`：仅输出
 - `BOTH`：同时支持输入和输出

---

# 示例
```java
/**
 * 定义 Item 类型的 IO 模式
 * 
 * @return
 */
@Override
protected IOMode getItemIO() {
	return IOMode.INPUT;
}

/**
 * 定义 Fluid 类型的 IO 模式
 * 
 * @return
 */
@Override
protected IOMode getFluidIO() {
	return IOMode.OUTPUT;
}

/**
 * 定义 Energy 类型的 IO 模式
 * 
 * @return
 */
@Override
protected IOMode getEnergyIO() {
	return IOMode.BOTH;
}
```
