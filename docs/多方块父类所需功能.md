# `IO`类型
 - `Item`
 - `Fluid`
 - `Energy`

# 定义`IO`模式
 - 通过方法重写的方式来选择每种类型的`IO`
 - 通过返回的数组来定义是否开启`I`和`O`
 - 第一个元素表示`I`, 第二个元素表示`O`
 - 如果数组内的元素为`true`则开启`I/O`, 如果为`false`则关闭`I/O`
 - 示例:
```java
/**
 * 定义Item类型的IO模式
 * 
 * @return
 */
@Override
protected boolean[] getItemIO() {
	return new boolean[]{true, false};
}

/**
 * 定义Fluid类型的IO模式
 * 
 * @return
 */
@Override
protected boolean[] getFluidIO() {
	return new boolean[]{false, false};
}

/**
 * 定�义Energy类型的IO模式
 * 
 * @return
 */
@Override
protected boolean[] getEnergyIO() {
	return new boolean[]{true, true};
}
```
