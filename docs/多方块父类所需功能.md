# `IO`类型

- `Item`
- `Fluid`
- `Energy`

---

# 定义 `IO` 模式

- 通过方法重写的方式来选择每种类型的 `IO`
- 通过返回 `IOMode` 来定义该类型的输入输出模式

---

# `IOMode` 说明

- `INPUT`: 仅输入
- `OUTPUT`: 仅输出
- `BOTH`: 同时支持输入和输出

---

# 示例

```java
/**
 * 定义 Item 类型的 IO 模式
 *
 * @return
 */
@Override
protected IOMode getItemIO() {
	return IOMode.INPUT;
}

/**
 * 定义 Fluid 类型的 IO 模式
 *
 * @return
 */
@Override
protected IOMode getFluidIO() {
	return IOMode.OUTPUT;
}

/**
 * 定义 Energy 类型的 IO 模式
 *
 * @return
 */
@Override
protected IOMode getEnergyIO() {
	return IOMode.BOTH;
}
```

---

# 定义槽位 / 容量

- 通过方法重写的方式来定义各类型的槽位或容量
- 不同类型的定义方式不同:
- `Item`: 槽位数量
- `Fluid`: 每个槽位数量 + 每个槽容量(依次定义)
- `Energy`: 总容量

# 示例

```java
/**
 * 定义 Item 类型的槽位数量
 * 
 * @return
 */
@Override
protected int getItemSlots() {
	return 8;
}

/**
 * 定义 Fluid 类型的槽位(mB)
 * FluidSlots(amount, mode, filter?)
 * 
 * @return
 */
@Override
protected FluidSlots[] getFluidSlots() {
	return new FluidSlots[]{
			// 从上到下, 从0开始计
			new FluidSlots(1000, IOMode.INPUT),
			new FluidSlots(4000, IOMode.OUTPUT),
			new FluidSlots(8000, IOMode.BOTH)
			/*
			 * 可选的过滤设计, 不写过滤默认允许所有流体
			 * new FluidSlots(1000, IOMode.BOTH, FluidFilters.only(Fluids.WATER))
			 * new FluidSlots(4000, IOMode.BOTH, FluidFilters.any())
			 * new FluidSlots(8000, IOMode.BOTH, FluidFilters.tag(FluidTags.WATER))
			 */
	};
}

/**
 * 定义 Energy 容量 (FE)
 * 
 * @return
 */
@Override
protected int getEnergyCapacity() {
	return 10000;
}
```
